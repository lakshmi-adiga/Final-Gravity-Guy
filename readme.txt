Title: Gravity Guy

Description: The player maneuvers a character to run and jump between on two platforms,
one above the other to avoid obstacles. The player has three lives. The longer the player
lasts without losing all 3 lives or going beyond the left edge of the screen, the higher their
score. When the game ends, if the player has acheived a highscore, the score will be 
displayed along with the username they inputted at the beginning.

How to run the program: Run the main.py file in an editor to load the game.

Libraries: No external libraries were used. cmu_112_graphics.py is required to run the game.

Shortcut Commands: N/A
